# DecisionesRapidas

Pequeña PWA sin backend para tomar decisiones rápidas entre sí y no.

## Uso

1. Abre el archivo `index.html` en un navegador o sirve la carpeta con `python -m http.server`.
2. Pulsa el botón **Decidir** para iniciar la animación de suspense y obtener una respuesta.
3. Instálala como app en tu dispositivo o úsala sin conexión (service worker incluido).
