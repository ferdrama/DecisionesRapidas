# DecisionesRapidas

PWA sin backend para tomar decisiones rápidas con dos modos: sí/no y dado de seis caras.

## Uso

1. Abre el archivo `index.html` en un navegador o sirve la carpeta con `python -m http.server`.
2. Usa el menú de hamburguesa para elegir el modo: **Sí / No** o **Dado de seis caras**.
3. Pulsa el botón **Decidir** para iniciar la animación de suspense y obtener una respuesta.
4. Instálala como app en tu dispositivo o úsala sin conexión (service worker incluido).
